RotamerEnsemble

class chilife.RotamerEnsemble(res, site=None, protein=None, chain=None, rotlib=None, **kwargs)

Create new RotamerEnsemble object.

name

String identifying RotamerEnsemble. Defaults to [nataa][site][res]_[chain] but chan be changed by the user if desired. This string will be used to name structures when saving PDBs.

Type:

str

res

3-character name of desired residue, e.g. R1A.

Type:

string

site

Protein residue number to attach library to.

Type:

int

protein

Object containing all protein information (coords, atom types, etc.)

Type:

MDAnalysis.Universe, MDAnalysis.AtomGroup

protein_tree

K-dimensional tree of the protein coordinates used internally for fast neighbor and distance calculations.

Type:

scipy.spatial.cKDTree

chain

Protein chain identifier to attach spin label to. Defaults to ‘A’ if no protein is provided.

Type:

str

coords

Cartesian coordinates of each rotamer in the ensemble.

Type:

numpy.ndarray

internal_coords

List of internal coordinate objects for each rotamer in the ensemble.

Type:

List[ProteinIC]

atom_types

Array of atom types, usually just element symbols, of each atom in the rotamer.

Type:

np.ndarray

atom_names

Array of atom names for each atom in the rotamer.

Type:

np.ndarray

selstr

Selection string that can be used with MDAnalysis select_atom method to select the site that the RotamerEnsemble is attached to.

Type:

str

input_kwargs

Stored copy of the keword arguments used when the object was created. input_kwargs is generally used for generating similar RotamerEnsembles.

Type:

dict

eval_clash

Boolean argument to instruct the RotamerEnsemble to evaluate clashes and trim rotamers on construction. If False The starting RotamerEnsemble is attached to the specified site and no clashes are evaluated. Post-construction clash evaluations can be performed using the evaluate method. Defaults to False.

Type:

bool

energy_func

Desired energy function to be used by the evaluate method when called by the user or when eval_clash is set to True. energy_func must be a callable object and accept two arguments: protein and ensemble Deafults to chiLife.get_lj_rep.

Type:

Callable

clash_ignore_idx

Array of atom indices of the protein attribute that should be ignored when the evaluate method is invoked. i.e. the atom indices of the native residue the RotamerEnsemble is being attached to.

Type:

np.ndarray

forgive

The “forgive-factor” to be used by the energy_func when the evaluate meth is invoked. If energy_func does not use a forgive-factor this value will be ignored. Defaults to 1.0.

Type:

float

temp

Temperature (in Kelvin) to use when re-evaluating rotamer weights using energy_func. Defaults to 298 K.

Type:

float

clash_radius

Cutoff distance (angstroms) for inclusion of atoms in clash evaluations. This distance is measured from clash_ori Defaults to the longest distance between any two atoms in the rotamer ensemble plus 5 angstroms.

Type:

float

alignment_method

Method to use when attaching or aligning the rotamer ensemble backbone with the protein backbone. Defaults to 'bisect' which aligns the CA atom, the vectors bisecting the N-CA-C angle and the N-CA-C plane.

Type:

str, callable

dihedral_sigmas

Standard deviations of dihedral angles (degrees) for off rotamer sampling. Can be a single numebr for isotropic sampling, a vector to define each dihedral individually or a matrix to define a value for each rotamer and each dihedral. Defaults to 35 degrees)

Type:

float, numpy.ndarray

weighted_sampling

Determines whether the rotamer library is sampled uniformly or based off of their intrinsic weights to generate the RotamerEnsemble. Defaults to False.

Type:

bool

use_H

Determines if hydrogen atoms are used or not. Defaults to False.

Type:

bool

atom_energies

Per atom energy (kcal/mol) value calculated suing self.energy_func.

Type:

numpy.ndarray

partition

Partition function value indicative of how much the ensemble has changed with respect to the original rotamer library. Only useful when not performing sampling.

Type:

float
